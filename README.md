# Boilerplate for website development

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# Available layouts

## Holy grail

- Landing page type
- Topbar (64px) + Main content centered (clamped between 800px and 1200px)
- Vertical scrolling

## App shell

- Dashboard type
- Topbar (64px) + Sidebar (240px) + Main content (1200px)
- Sidebar hidden when browser width < 1440px
